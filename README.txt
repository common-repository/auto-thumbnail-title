Este plugin permite colocar el titulo sobre la imagen destacada de las entradas

== Description ==

Con este plugin cada vez que se muestre la imagen destacada de una entrada (en loop o single) se imprimira en la esquina inferior derecha de la misma el titulo de la entrada en texto pudiendo seleccionar desde el administrador el clor de fondo asi como el de la fuente.

Tenga en cuenta que el titulo se muestra dentro de un elemento h1 y que responde a los estilos que ya contenga su plantilla para este elemento.

== Installation ==

1. Instale el plugin "Auto thumbnail title" en su sitio como cualquier otro plugin.
2. Active el plugin en el menu 'Plugins' de WordPress.
3. Configure los estilos en el menu ATT que se a creado en el administrador.
